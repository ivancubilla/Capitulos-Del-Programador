Nuestro Primer Proyecto

Muy bien ya estamos preparados? Vamos a crear nuestro Primer Proyecto para eso, abrimos nuestro Intellij y seleccionamos Nuevo Proyecto (New Proyect)



Le indicamos que va a ser un Proyecto JAVA



Y seleccionamos la version de nuestro JDK que descargamos, usualmente ya nos suele reconocer automaticamente, de igual manera si no es el caso, seleccionan >>ADD JDK<< y buscan la carpeta que descargaron. Despues le daran en NEXT



Por el momento no crearemos ningun template, asique saltearemos este paso.
Le daremos a NEXT para continuar



Elegiremos un nombre para Nuestro Proyecto y en donde se va a localizar. Por ultimo le daremos a finalizar (finish)
